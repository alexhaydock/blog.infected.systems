# blog.infected.systems

Content directory for my blog: https://blog.infected.systems

### Catpuccin-based Colour Scheme
* Light Mode (Default): Latte
* Dark Mode: Frappé
* Code Blocks: Mocha

### Test site locally
```sh
hugo serve
```

### Deploy site
This script builds and syncs the site to Nintendo Wii:
```sh
./build-and-sync.sh
```
